# How to Install the AURA Connector Extension

1.  Open Google Chrome (or Edge/Brave).
2.  Navigate to `chrome://extensions`.
3.  Enable **Developer mode** (toggle in the top right corner).
4.  Click **Load unpacked**.
5.  Select this folder: `/Users/indra/Documents/aura-project/chrome_extension`.
6.  The extension "AURA Connector" will appear.

## How to Use
1.  Ensure your AURA server is running (`localhost:8000`).
2.  Go to **ChatGPT.com**.
3.  Click the extension icon in your browser toolbar and toggle it **ON**.
4.  Refresh the ChatGPT page.
5.  You will see a purple **"🗣️ AURA"** button on every ChatGPT response.
6.  Click it, and your Avatar will read the message!
